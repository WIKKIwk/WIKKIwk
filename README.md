# Abdulfattox

I build industrial systems that actually run businesses, not just code.

## What I Build

- ERPNext integrations
- Mobile bridge systems
- Telegram automation
- RFID, Zebra, scales, and printer workflows
- AI assistants for real operations

## Main Product Groups

### Mobile Group

- `erpnext-mobile-bridge` - bridge between ERPNext and the mobile app
- `erpnext-mobile-client` - mobile client for ERP operations
- `erpnext-bridge-fields` - ERPNext module for clean server communication
- `erpnext-bot-bootstrap` - starter controller for ERPNext bot setup

### Hardware And Operations

- `zebra-scale-erpnext-core` - Zebra RFID printer and scale integration
- `rfid-zebra-scale-hybrid-core` - hybrid core for UHF, Zebra, and scale hardware
- `uhf-rfid-java-cli` - Java CLI for UHF reader control
- `zebra-rfid-bridge-core` - standalone Zebra bridge
- `printer-scanner-telegram-core` - Telegram-controlled printer and scanner bot
- `industrial-scale-simulator` - simulator for testing scale workflows

### ERPNext Modules

- `erpnext-ui-theme-suite` - customizable ERPNext theme
- `erpnext-security-suite` - ERPNext security module
- `erpnext-admin-ai-assistant` - admin-only AI chat inside ERPNext
- `erpnext-guided-ai-assistant` - guided ERPNext tutor and assistant

### Web And AI

- `abdulfattox-web` - personal site
- `commerce-web-platform` - customizable ecommerce project
- `ai-sheets-platform` - AI-powered sheets platform
- `expert-answer-ai-agent` - expert-style Telegram AI bot
- `persona-companion-agent` - clone-style Telegram bot
