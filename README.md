# Abdulfattox

I build industrial systems that actually run businesses, not just code.

## What I Build

- ERPNext integrations
- Mobile bridge systems
- Telegram automation
- RFID, Zebra, scales, and printer workflows
- AI assistants for real operations

## Main Product Groups

### Mobile Group

- [`erpnext-mobile-bridge`](https://github.com/WIKKIwk/erpnext-mobile-bridge) - bridge between ERPNext and the mobile app
- [`erpnext-mobile-client`](https://github.com/WIKKIwk/erpnext-mobile-client) - mobile client for ERP operations
- [`erpnext-bridge-fields`](https://github.com/WIKKIwk/erpnext-bridge-fields) - ERPNext module for clean server communication
- [`erpnext-bot-bootstrap`](https://github.com/WIKKIwk/erpnext-bot-bootstrap) - starter controller for ERPNext bot setup
