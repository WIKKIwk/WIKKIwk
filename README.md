# Abdulfattox

I build industrial systems that actually run businesses, not just code.

## What I Build

- ERPNext integrations
- Mobile bridge systems
- Telegram automation
- RFID, Zebra, scales, and printer workflows
- AI assistants for real operations

## Main Product Groups

### Mobile Group

- `mobile_server` - bridge between ERPNext and the mobile app
- `erp-mobile-suite` - mobile client for ERP operations
- `customfield_for_server` - ERPNext module for clean server communication
- `TLCU` - starter controller for ERPNext bot setup

### Hardware And Operations

- `zebra-scale-integration` - Zebra RFID printer and scale integration
- `uhfr-zb-integration-core` - hybrid core for UHF, Zebra, and scale hardware
- `uhf-rfid-cli` - Java CLI for UHF reader control
- `erpnext-zebra-bridge` - standalone Zebra bridge
- `printer_bot` - Telegram-controlled printer and scanner bot
- `fake-simulator` - simulator for testing scale workflows

### ERPNext Modules

- `ERPNext_theme` - customizable ERPNext theme
- `erp_suite` - ERPNext security module
- `ERPNext_ai_integration_app` - admin-only AI chat inside ERPNext
- `erpnext-tutor-ai-assistant` - guided ERPNext tutor and assistant

### Web And AI

- `ME` - personal site
- `Ecommer_web` - customizable ecommerce project
- `Sheets_W12` - AI-powered sheets platform
- `Genius` - expert-style Telegram AI bot
- `Companion` - clone-style Telegram bot
