## Status Legend

- `core` - main engine of a product family
- `bridge` - connector between systems
- `module` - ERPNext extension or add-on
- `hardware` - project that talks to physical devices
- `app` - user-facing application or bot
- `tool` - utility or helper project
- `experimental` - test or prototype
- `deprecated` - no longer active
